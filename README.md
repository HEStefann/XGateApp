## CRUD with Authentication

Requirements:
PHP 8.1+

Composer

Laravel 12

MySQL

**Installation Steps:**
    git clone https://github.com/your-username/XGateApp.git
    cd XGateApp

Install Dependencies: 
    composer install
    npm install

Configure Environment
    cp .env.example .env

Generate App Key
    php artisan key:generate

Run Migrations and Seed Database
    php artisan migrate --seed

Start the Server
    php artisan serve
